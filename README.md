# Creator Preview Plugin

提供编辑器中内嵌预览功能。<br>
目前支持 1.4.0 Beta9 及以上版本的 Creator

## 插件安装方法

请参考 [扩展编辑器:安装与分享](http://www.cocos.com/docs/creator/extension/install-and-share.html) 文档。
